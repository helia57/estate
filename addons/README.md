## Odoo Hello World Module

Ce dépôt contient un module Odoo 19 simple appelé `hello_world`.

## Fonctionnalités

- Nouveau modèle `hello.world`
- Vue liste + vue formulaire
- Menu dédié dans le backend
- Action ouvrant les enregistrements

## Installation

1. Copier le module dans le dossier `addons`
2. Redémarrer Odoo
3. Mettre à jour la liste des modules
4. Installer *Hello World*
